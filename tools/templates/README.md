# Templates

This directory is reserved for future `captool create` and scaffold/template assets.

Current status:

- no active template flow is implemented yet
- this directory exists to make the future toolchain shape explicit

Design intent:

- keep scaffold/template assets close to `captool`
- avoid mixing future template assets into `tests/`, `src/`, or `demo/`
- let `captool create` grow naturally without reshaping the whole `tools/` tree later
